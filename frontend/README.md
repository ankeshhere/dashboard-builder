# Dashboard Builder

A plugin-based dashboard builder inspired by Power BI and Grafana.

## Tech Stack

Frontend

- React
- TypeScript
- Vite
- Material UI
- Zustand

Backend (planned)

- Quarkus
- PostgreSQL

## Current Status

Current milestone: 4.1

Implemented:

- Theme
- Plugin Registry
- Dashboard Store
- Sidebar
- Canvas
- Add Text Widget

## Run

npm install
npm run dev

## Workflow

Each milestone:

1. Build feature
2. Verify
3. Commit
4. Push
